Classpublic class SoundConductor
InheritanceSoundConductor Object

SoundConductor クラスは、様々なサウンドの再生を統括・管理するための静的なクラスです.

このクラスには、以下の機能が実装されています。

  • SoundGenerator 機能を使用した、始点・終点を指定した無限ループ機能
  • サウンドのグルーピング機能
  • ボリュームのフェードイン・アウト
  • 「マスターボリューム」「グループボリューム」、そして「音そのもののボリューム」を使用した多段性のボリューム調整機能
  • ポーズ・リジューム機能

initialize()method 
public static function initialize(useSharedSoundGenerator:Boolean = false, bufferType:SoundBufferType = null):void

SoundConductor の初期化を行います.

使用する際には必ずこのメソッドを実行して初期化してください。

Parameters

useSharedSoundGenerator:Boolean (default = false) — SoundGenerator 機能を使用するかを指定します。 この値を true にすると、サウンド再生方式の種類として SoundPlayType.SINGLE_SOUND_GENERATOR と SoundPlayType.SHARED_SOUND_GENERATOR が使用できるようになります。 デフォルトは false です。
 
bufferType:SoundBufferType (default = null) — SoundGenerator 機能で扱われるサウンドのバッファを保持する量を設定します。 useSharedSoundGenerator が false の場合、この値を設定する必要はありません。 設定には SoundBufferType クラスの定数を使用してください。デフォルトは null です。

playSoundObject()method 
public static function playSoundObject(sound:Sound, groupName:String, startTime:Number = 0, loops:int = 0, volume:Number = 1, pan:Number = 0):SoundChannel

Sound オブジェクトを通常の再生方法で再生します.

このメソッドを通して再生することにより、SoundConductor が管理しているマスターボリュームとグループボリュームの恩恵を得ることが出来ます。 グループを指定した場合、そのグループのミュート状態も適用されます。

ミュート状態はボリュームを 0 にすることで再現しているため、ミュートの解除はボリュームを上げることで対応する必要が有ります。

Parameters

sound:Sound
 
groupName:String
 
startTime:Number (default = 0)
 
loops:int (default = 0)
 
volume:Number (default = 1)
 
pan:Number (default = 0)

Returns
SoundChannel — 再生した Sound から生成される SoundChannel クラスです。何らかの問題で再生されなかった場合は null が返ります。
